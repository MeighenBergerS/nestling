# Plotting

This track teaches you to create clear, publication-quality scientific figures.
You will learn the principles of good data visualisation and how to apply them
using matplotlib and a custom style file.

!!! note "Under construction"
    Lesson content is being developed. The outline below shows what each lesson will cover.

## Lessons

- **Lesson 01 — Figure Design Principles**: What makes a good scientific figure: clarity, honesty, and accessibility.
- **Lesson 02 — matplotlib Basics**: Creating line plots, scatter plots, histograms, and multi-panel figures.
- **Lesson 03 — Publication Style**: Using the Nestling matplotlib style, LaTeX labels, and exporting to PDF.
- **Lesson 04 — Colour and Accessibility**: Choosing colourblind-friendly palettes and avoiding common pitfalls.

## Examples

See the [`examples/plotting/`](https://github.com/MeighenBergerS/nestling/tree/main/examples/plotting) folder
for runnable notebooks and scripts demonstrating basic and publication-quality plots.

The Nestling matplotlib style file is at `examples/plotting/styles/nestling.mplstyle`.
