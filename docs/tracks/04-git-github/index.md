# Git & GitHub

This track takes you from zero to confident with version control.
You will learn the core Git workflow, how to collaborate on GitHub, and how to use
pull requests and issues effectively in a research project.

!!! note "Under construction"
    Lesson content is being developed. The outline below shows what each lesson will cover.

## Lessons

- **Lesson 01 — Git Basics**: Initialising a repository, staging, committing, and reading history.
- **Lesson 02 — Branching**: Creating and merging branches, resolving conflicts.
- **Lesson 03 — GitHub Collaboration**: Forks, pull requests, and code review.
- **Lesson 04 — Issues and Project Management**: Tracking work and linking issues to commits.
- **Lesson 05 — GitHub Actions**: Automating tests and checks on every push.
