# Nestling

Welcome to **Nestling**, a beginner-friendly, physics-first learning repository for early-career researchers and graduates.

Nestling provides short guides, templates, examples, and exercises to help you build practical research skills quickly.
Every track is self-contained — pick the one most relevant to where you are right now.

## Learning Tracks

| Track | What you will learn |
|---|---|
| [ArXiv & Reading Papers](tracks/01-arxiv/index.md) | Finding, reading, and managing research papers |
| [Journal Club](tracks/02-journal-club/index.md) | Presenting and discussing papers with peers |
| [Writing Papers](tracks/03-writing/index.md) | Structure, style, and LaTeX for physics papers |
| [Git & GitHub](tracks/04-git-github/index.md) | Version control and collaborative workflows |
| [Coding Practices](tracks/05-coding/index.md) | Project layout, documentation, and testing |
| [Plotting](tracks/06-plotting/index.md) | Scientific visualisation with matplotlib |
| [Working with LLMs](tracks/07-llm-practices/index.md) | Using language models responsibly in research |
