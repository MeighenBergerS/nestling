# FAQ

!!! note "Under construction"
    This FAQ is being expanded. Suggest questions via a GitHub issue.

---

**Do I need to complete the tracks in order?**

No. Each track is self-contained. Start with the one most relevant to your current work.
The [Getting Started](getting-started/how-to-use-this-repo.md) guide can help you decide.

---

**What Python version do I need?**

Python 3.11 or later. See [Environment Setup](getting-started/environment-setup.md) for installation instructions.

---

**Can I use these materials in a course or workshop?**

Yes. Nestling is licensed under GPL-3.0-or-later. You are free to use, adapt, and redistribute
the materials provided you follow the licence terms and attribute the source.

---

**I found an error. How do I report it?**

Open an issue on the [GitHub repository](https://github.com/MeighenBergerS/nestling/issues).
Include the page, the error, and (if you know it) the correction.

---

**How do I contribute a lesson?**

See `CONTRIBUTING.md` in the repository root for the full contribution guide.
