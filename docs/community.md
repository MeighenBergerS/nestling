# Community

## Discussions

Have a question, idea, or something to share?
Join the conversation on [GitHub Discussions](https://github.com/MeighenBergerS/nestling/discussions).

| Category | Use it for |
|---|---|
| [Q&A](https://github.com/MeighenBergerS/nestling/discussions/categories/q-a) | Questions about lessons, examples, or setup |
| [Ideas](https://github.com/MeighenBergerS/nestling/discussions/categories/ideas) | New lesson proposals before opening an issue |
| [Show and Tell](https://github.com/MeighenBergerS/nestling/discussions/categories/show-and-tell) | Share plots, projects, or work made using the lessons |
| [General](https://github.com/MeighenBergerS/nestling/discussions/categories/general) | Anything else |

## Contributing

We welcome contributions: new lessons, examples, bug fixes, and documentation improvements.
See the [contribution guidelines](https://github.com/MeighenBergerS/nestling/blob/main/CONTRIBUTING.md)
to get started.

## Reporting Issues

Found a bug or want to suggest a change?
[Open an issue](https://github.com/MeighenBergerS/nestling/issues/new/choose) using one of the
structured templates.
